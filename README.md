# RandomPassword
Автор: Бральгин Даниил
Описание: Программа для генерации надежных паролей с настройкой параметров. Создает простые или сложные пароли, варьируя настройки использования букв, цифр и спецсимволов. Алфавит для цифр - десятичная система счисления. Символы прописаны строкой, а буквы используются из латинского алфавита в строчной и прописной форме.
Примеры использования: 
1. Базовый (Пин-код)
Настройки: Длина 4, только «Цифры».
Результат: В истории появится запись вида 5829
2. Безопасный (Для соцсетей)
Настройки: Длина 12, «Буквы» + «Цифры».
Результат: Пароль формата x7K2pL9nQ1rW.
3. Максимальный (Для админа)
Настройки: Длина 32, все чекбоксы (Буквы + Цифры + Спецсимволы).
Результат: Сложная строка вроде j#8$K1!pL9*qR2@vN5^zX7&mB4(tY0_s.
